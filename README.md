# HYEWON의 전처리 과정

1. '불임 원인 - 여성 요인', '난자 채취 경과일' 모두 컬럼 제거 (유니크 값이 1개인 경우 컬럼 삭제)

2. '배란 자극 여부' == 0이고 '배란 유도 유형' == '알 수 없음'인 경우 (전부 그랬음) '배란 유도 유형'을 '시행 없음'으로 채우기

3. '배란 자극 여부' == 1이고 '배란 유도 유형' == '알 수 없음'인 경우 '배란 유도 유형'을 '기록되지 않은 시행'으로 채우기

4. ['총 시술 횟수', '클리닉  내 총 시술 횟수', 'IVF 시술 횟수', 'DI 시술 횟수', '총 임신 횟수', 'IVF 임신 횟수', 'DI 임신 횟수', '총 출산 횟수', 'IVF 출산 횟수', 'DI 출산 횟수']는 nan이 없고 10회를 넘는 값이 없으므로
   모두 첫 글자만 슬라이싱해서 int로 바꾸기

5. '시술 유형' == 'IVF'이고 '특정 시술 유형' == nan인 경우 '특정 시술 유형'을 'Unknown'으로 채우기

6. '시술 유형' == 'DI'이고 '특정 시술 유형' == 'IUI'이고 '단일 배아 이식 여부' == nan인 경우 '단일 배아 이식 여부'를 0으로 채우기 

7. '특정 시술 유형' == 'IUI'인 경우 '착상 전 유전 검사 사용 여부'와 '착상 전 유전 진단 사용 여부'를 모두 0으로 채우기

8. '이식된 배아 수' == 0이고 '특정 시술 유형' == 'IUI'인 경우는 배아를 생성하거나 이식하지 않는 시술 방식이므로 이 경우에는 '이식된 배아 수를 모두 0으로 채우기 -> 이 경우 배아 수 관련 다른 속성에도 적용할 수 있을 것 

9. '특정 시술 유형' == 'IUI'인 경우 '배아 생성 주요 이유'가 전부 nan이다. 이는 이 시술이 배아를 생성하지 않는 시술이기 때문에 이 경우, '배아 생성 주요 이유'를 모두 '배아 생성하지 않음'으로 채우기
